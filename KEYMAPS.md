# Neovim Keymaps (LazyVim)

`<leader>` = `Space`

## Buffer

| キー | 動作 |
|------|------|
| `H` | 前のバッファ |
| `L` | 次のバッファ |
| `<leader><space>` | バッファ一覧から選択 (telescope) |
| `<leader>,` | バッファ一覧から選択 (telescope) |
| `<leader>bb` | 直前のバッファに戻る |
| `<leader>bd` | バッファを閉じる |
| `<leader>bD` | 他のバッファを全て閉じる |
| `<leader>b1` 〜 `<leader>b9` | バッファ番号で直接ジャンプ |

## ファイル操作

| キー | 動作 |
|------|------|
| `<leader>ff` | ファイル検索 (telescope) |
| `<leader>fg` | git管理ファイルの検索 (telescope) |
| `<leader>fr` | 最近開いたファイル |
| `<leader>/` | grep検索 |
| `<leader>e` | ファイルツリー (neo-tree) |
