# Neovim Keymaps (LazyVim)

`<leader>` = `Space`

## Buffer

| キー | 動作 |
|------|------|
| `H` | 前のバッファ |
| `L` | 次のバッファ |
| `<leader><space>` | バッファ一覧から選択 (telescope) |
| `<leader>,` | バッファ一覧から選択 (telescope) |
| `<leader>bb` | 直前のバッファに戻る |
| `<leader>bd` | バッファを閉じる |
| `<leader>bD` | 他のバッファを全て閉じる |
| `<leader>b1` 〜 `<leader>b9` | バッファ番号で直接ジャンプ |

## ファイル操作

| キー | 動作 |
|------|------|
| `<leader>ff` | ファイル検索 (telescope) |
| `<leader>fg` | git管理ファイルの検索 (telescope) |
| `<leader>fr` | 最近開いたファイル |
| `<leader>/` | grep検索 |
| `<leader>e` | ファイルツリー (neo-tree) |

## Neo-tree 内操作

### 移動・開閉
| キー | 動作 |
|------|------|
| `<space>` / `<cr>` | ノードを開く / ディレクトリを展開 |
| `C` | ディレクトリを閉じる |
| `z` | 全ノードを閉じる |
| `<bs>` | 親ディレクトリへ移動 |

### 検索・フィルター
| キー | 動作 |
|------|------|
| `/` | fuzzy検索 (インクリメンタル) |
| `f` | フィルター入力して確定 |
| `<C-x>` | フィルタークリア |
| `H` | 隠しファイルの表示トグル |

### ファイル操作
| キー | 動作 |
|------|------|
| `a` | ファイル作成 |
| `A` | ディレクトリ作成 |
| `d` | 削除 |
| `r` | リネーム |
| `y` | コピー (クリップボード) |
| `x` | カット (クリップボード) |
| `p` | ペースト |
| `c` | コピー (パス入力) |
| `m` | 移動 (パス入力) |

### 表示・その他
| キー | 動作 |
|------|------|
| `P` | プレビュー |
| `S` | 横分割で開く |
| `s` | 縦分割で開く |
| `t` | タブで開く |
| `R` | 更新 |
| `q` | neo-tree を閉じる |
| `?` | ヘルプ |
